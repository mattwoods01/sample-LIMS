Updates Labguru maintenances and provides maintenance sheet to see current status of equipment maintenance


<u>**REQUIRED FIELDS:**</u>

1. **Name** - Name of equipment that is found on Labguru 

2. **Maintenance Type** - Must be one of the types that is saved on Labguru (Will add dropdown)

3. **Maintenance Frequency** - Must be an integer values (Will add dropdown)

4. **Maintenance Date** - Must be formatted as YYYY-MM-DD.

<u>**HOW TO USE (Updating Maintenances):**</u>

1. First, either upload or manually type in the information to be used for updating maintenances on Labguru.
    
    1. If uploading, the file must be an excel file with at least the 'Name' and 'Calibration Information' found within the headers.  Once complete, click the 'Choose File' button to open the file explorer to choose your file.
    
    2. If manually typing, you must make sure that the instrument name exists within Labguru and the 'Calibration Information' field is in the correct format.

2. Once your file is chosen, click on the 'Upload' button to update the webpage with the file contents.

3. Once you have made the proper adjustments, click 'Run' to initiate the updating process.

4. Once complete, you will see a notification saying 'Finished updating maintenances'.  You can check if the update was successful by looking at the labguru in

<u>**HOW TO USE (Downloading Maintenance Sheet):**</u>

1. To download the Maintenance sheet click the 'Download Maintenance Sheet' button

2. Your file will take a few seconds to download, but once 'Sending File data...' is seen in the text box, your file will be downloaded to the client

<u>**HOW TO USE (Creating new maintenance type on Labguru):**</u>

1. Go to: [*Maintenance Log*](https://thermofisher.labguru.com/system/instruments/maintenance_log)

2. Click the 'Edit' button next to 'Maintenance type'

3. Click 'Add maintenance type' and type in a uniquely named maintenance type

4. Click 'Save'
